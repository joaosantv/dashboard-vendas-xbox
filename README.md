# 🎮 Dashboard de Vendas - Ecossistema Xbox

## 📊 Sobre o Projeto
Este projeto foi desenvolvido como o desafio prático final do bootcamp **TOTVS + DIO**, com o objetivo de transformar dados brutos de assinaturas em uma ferramenta estratégica de Business Intelligence. O dashboard permite analisar o faturamento por tipo de plano e a eficácia de renovações automáticas, facilitando a tomada de decisão rápida.

## 🛠️ Tecnologias e Conceitos Aplicados
* **Microsoft Excel:** Utilizado para todo o processo de ETL (Extração, Transformação e Carga).
* **Tabelas Dinâmicas:** Estruturação de dados na aba de "Cálculos" para sumarizar milhares de registros de vendas.
* **Gráficos Dinâmicos:** Visualizações interativas que respondem em tempo real aos filtros aplicados.
* **Segmentação de Dados (Slicers):** Implementação de botões de filtro para análise granular por tipo de assinatura e renovação.
* **Design UI/UX:** Interface personalizada com o "Modo Noturno" e identidade visual da marca Xbox, focando em clareza e usabilidade.

## 📂 Estrutura do Arquivo
O arquivo está organizado seguindo as melhores práticas de organização de dados:
1. **Bases:** Dados brutos de assinaturas.
2. **Assets:** Logos e paleta de cores hexadecimais para manter a consistência visual.
3. **Cálculos:** O "motor" do dashboard, onde residem as tabelas dinâmicas.
4. **Dashboard:** A interface final interativa para o usuário.

## 🚀 Como Visualizar
1. Faça o download do arquivo `.xlsx` presente neste repositório.
2. Abra no Microsoft Excel (recomenda-se habilitar a edição para o funcionamento dos filtros).
3. Interaja com os botões de segmentação para explorar os dados.
